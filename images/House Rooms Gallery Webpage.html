Creating a simple webpage to showcase photos of house rooms using GitHub Pages involves several steps. Heres a step-by-step guide to help you through the process:

Step 1: Set Up Your GitHub Repository

  1. Create a GitHub Account: If you don't already have one, go to GitHub and sign up.
  2. Create a New Repository:
    • Click the "+" icon in the upper right corner and select "New repository".
    • Name your repository (e.g., house-rooms-gallery).
    • Add a description if you like (e.g., "A simple webpage showcasing photos of house rooms").
    • Choose "Public" and check "Initialize this repository with a README".
    • Click "Create repository".

Step 2: Add Your HTML, CSS, and Images

  1. Clone the Repository: Clone the repository to your local machine using Git or GitHub Desktop.
    sh
    git clone https://github.com/your-username/house-rooms-gallery.git
  2. Create HTML File: Create an index.html file with the following content:
    html
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>House Rooms Gallery</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            display: flex;
            flex-direction: column;
            align-items: center;
            background-color: #f0f0f0;
            margin: 0;
            padding: 0;
        }
        h1 {
            margin: 20px 0;
        }
        .gallery {
            display: flex;
            flex-wrap: wrap;
            justify-content: center;
        }
        .gallery img {
            margin: 10px;
            border: 2px solid #ddd;
            border-radius: 4px;
            width: 300px;
            height: auto;
        }
    </style>
</head>
<body>

    </div>
</body>
</html>
  3. Add Images: Create an images directory in your repository and add your photos (living-room.jpg, kitchen.jpg, bedroom.jpg, bathroom.jpg, etc.) into this directory.

Step 3: Push Your Changes to GitHub

  1. Add and Commit Your Changes:
    sh
    git add .
git commit -m "Initial commit with HTML and images"
  2. Push to GitHub:
    sh
    git push origin main

Step 4: Enable GitHub Pages

  1. Go to Repository Settings:
    • Navigate to your repository on GitHub.
    • Click on "Settings".
  2. Enable GitHub Pages:
    • Scroll down to the "GitHub Pages" section.
    • Under "Source", select "main" branch.
    • Click "Save".

Step 5: Access Your Webpage

After enabling GitHub Pages, your webpage should be available at:

Replace your-username with your GitHub username. It may take a few minutes for the page to be live.
